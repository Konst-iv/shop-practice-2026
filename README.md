# Интернет-магазин учебных товаров

Современное веб-приложение на Django для продажи учебных материалов и канцелярии.

## Инструкция по запуску

Для запуска проекта локально выполните следующие шаги:

### 1. Клонирование репозитория
Склонируйте проект к себе на компьютер и перейдите в его директорию:
```bash
git clone <ссылка_на_ваш_репозиторий>
cd <имя_папки_проекта>
```

### 2. Настройка виртуального окружения
Создайте и активируйте виртуальное окружение:


Для Windows:
```bash
    python -m venv venv
    venv\Scripts\activate
```

Для macOS / Linux:
``` bash
    python3 -m venv venv
    source venv/bin/activate
```

### 3. Установка зависимостей

Установите все необходимые библиотеки:

```bash
pip install -r requirements.txt
```

### 4. Применение миграций

Создайте структуру таблиц в базе данных:
```bash
python manage.py migrate
```
### 5. Создание суперпользователя (Администратора)

Чтобы иметь доступ к админ-панели Django, создайте аккаунт администратора:
```bash
python manage.py
```
Введите имя пользователя, почту и пароль (пароль при вводе отображаться не будет).
#6. Запуск сервера разработки

Запустите локальный сервер Django:
```bash
python manage.py runserver
```
Теперь проект доступен в браузере по адресу: http://127.0.0.1:8000/
Панель администратора находится по адресу: http://127.0.0.1:8000/admin/
